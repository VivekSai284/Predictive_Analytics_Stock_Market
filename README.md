# Predictive Analytics for Stock Market Movements

## Project Overview
This project focuses on predicting stock market price movements using machine learning techniques and historical stock market data.

The project includes:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Models
- Prediction Visualization
- Model Evaluation

---

## Objectives
- Analyze historical stock market data
- Identify trends and patterns
- Build predictive machine learning models
- Compare prediction accuracy

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## Dataset
Dataset Source:
- Yahoo Finance
- Kaggle Stock Market Dataset

Dataset includes:
- Date
- Open
- High
- Low
- Close
- Volume

---

## Machine Learning Models Used
1. Linear Regression
2. Random Forest Regressor

---

## Features Implemented
- Data Cleaning
- Missing Value Handling
- Moving Average Feature
- Stock Price Visualization
- Actual vs Predicted Comparison
- Model Performance Evaluation

---

## Evaluation Metrics
- Mean Squared Error (MSE)
- R2 Score

## How to Run

1. Install dependencies

```bash
pip install -r requirements.txt
```

2. Start Jupyter Notebook

```bash
jupyter notebook
```

3. Open notebook

```bash
Predictive_Analytics_Stock_Market.ipynb
```

The machine learning models successfully predicted stock price trends using historical stock data.

Random Forest showed better prediction accuracy compared to Linear Regression.
